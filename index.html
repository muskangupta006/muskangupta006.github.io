Projects

Arrhythmia Detection Model

During my internship at CytoHub, I developed a machine learning model to detect arrhythmias in cardiac organoids. The goal was to create a tool that could work directly with outputs from MuscleMotion, a video analysis software used in biomedical research. I designed and trained models using selected motion-based features, carefully preprocessing noisy and incomplete data to improve accuracy. Through iterative experimentation and validation, I built a model that provides reliable probability scores for arrhythmia detection, helping researchers identify irregular beating patterns earlier and more efficiently.

Muskan worked in my team as a Machine Learning Engineering Intern during Summer 2025. She successfully developed a sophisticated pipeline to detect arrhythmia in cardiac organoids, a project that showcased her impressive technical skills and problem-solving abilities. Her project involved a comprehensive approach, including preprocessing noisy biological signals, engineering time-series features, and training an XGBoost model using labeled ECG data.
